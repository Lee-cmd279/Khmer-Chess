# អុកចត្រង្គ — Ouk Chaktrang AI v5
## Complete Multi-Platform Training Setup

All 5 platforms (GitHub Actions, Render, HuggingFace, Kaggle, Colab)
feed into one JSONBin hub. Your game loads the latest weights automatically.

## Quick Start
1. Set up JSONBin (jsonbin.io) — get Key + Bin ID
2. Upload all files to GitHub → enable Pages
3. Add JSONBIN_KEY and JSONBIN_BIN as GitHub Secrets
4. Deploy ouk_train.py to Render.com
5. Open game → Admin → Configure Cloud → enter your keys

See SETUP.md for full step-by-step instructions.
